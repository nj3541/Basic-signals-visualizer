Key concepts

  • Unit step u(t) = 0 for t < 0, 1 for t ≥ 0 → basic causal building block.
  • Ramp r(t) = t·u(t) → line starting at 0 for t ≥ 0.
  • Sinusoid A·sin(ωt) → periodic, fundamental frequency f = ω/(2π).
  • Exponential A·e-atu(t) → decays for a > 0, causal.
